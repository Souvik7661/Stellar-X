# 🧠 Stellar Split Project Mind Map

## 📌 Idea
Stellar Split (formerly SplitX) is a blockchain-based expense splitting app that allows users to split bills and settle debts using XLM on Stellar.

## 🚀 Features
- [x] Wallet connection
- [x] Balance display
- [ ] Add expense (support manual entry public keys)
- [ ] Debt calculation
- [ ] Send XLM (settle debt)
- [ ] Smart contract integration
- [ ] Expense history
- [ ] Dashboard
- [ ] Real-time updates
